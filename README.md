# Object-Oriented Programming Practice

## Quick Start

1. `git clone https://github.com/team-waff1e/Object.git`

## Conventions

1. Branch: `{english-name}`
2. Directory: `/src`
3. Package: `oop.{english-name}.{chapter##}`
   - e.g. `oop.yourname.chapter01`
4. Descriptions: Pull Requests
5. Questions: Issues
